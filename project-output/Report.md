## 技术方案摘要

### 后端技术栈

| 层级 | 技术选型 |
|-----|---------|
| **编程语言** | Go 1.22+ |
| **Web框架** | Gin |
| **ORM层** | GORM |
| **数据库** | PostgreSQL 15+ |
| **缓存** | Redis 7+ |
| **消息队列** | Redis Stream |
| **认证** | JWT + Refresh Token |
| **权限控制** | Casbin |

### 前端技术栈

| 层级 | 技术选型 |
|-----|---------|
| **核心框架** | React 18.x + TypeScript 5.x + Next.js 14.x |
| **UI组件库** | shadcn/ui + Tailwind CSS 3.x |
| **状态管理** | Zustand + React Query |
| **图表可视化** | Recharts + React Gantt |
| **表单管理** | React Hook Form + Zod |
| **构建工具** | Vite |

### 部署架构

- **环境划分**：开发环境、测试环境、预发布环境、生产环境
- **容器化**：Docker + Docker Compose
- **监控**：Prometheus + Grafana
- **日志**：ELK Stack
- **CI/CD**：自动化构建、自动化测试、自动化部署
