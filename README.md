# 项目一体化平台 · 贵州习酒

基于完整需求清单（60 个功能项）完整实现的企业级项目管理一体化平台，采用奢华精致的设计风格。

## ✨ 设计亮点

| 元素 | 选择 |
|------|------|
| **主色调** | 深邃蓝 (#0A0F1D) + 琥珀金 (#F59E0B) + 象牙白 (#FEFAF5) |
| **展示字体** | Bricolage Grotesque（独特、优雅） |
| **正文字体** | Crimson Pro（精致、易读） |
| **布局** | 不对称+层次感 |
| **动效** | 精心编排的页面加载动画、悬停微交互 |
| **视觉细节** | 渐变边框、背景纹理、柔滑阴影 |

## 💎 设计哲学

> **"Luxury/Refined"（奢华精致）**

- 不使用俗套的渐变，而是采用深邃背景搭配琥珀金点缀
- 不使用通用字体，而是选择有特色的 Bricolage Grotesque 和 Crimson Pro
- 不使用可预测的布局，而是采用不对称+层次化设计
- 不使用分散的微交互，而是精心编排高影响力的页面加载动画
- 不使用默认的背景，而是添加纹理、阴影、渐变边框等细节
