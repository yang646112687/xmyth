## 1. 文档概述

### 1.1 文档目的
本文档详细描述"项目管理一体化平台"的重新设计方案，包括 UI/UX 设计规范、交互说明、原型设计、非功能性需求等，作为前端开发团队的执行依据。

### 1.2 设计背景
根据《贵州省国资委监管企业数字化监管项目代码工作指引（试行）》要求，结合已有的《项目可行性分析报告》和《功能需求清单》，对系统进行重新设计。本次设计参考「企查查」的信息型界面设计，注重信息的清晰展示与高效检索。

### 1.3 关键设计决策

| 决策项 | 方案 | 说明 |
|---------|------|------|
| **设计风格** | 企查查风格 | 信息型界面，注重清晰展示与高效检索 |
| **主色调** | 君品绿 (#0A5C36) | 体现专业、可信、沉稳的品牌调性 |
| **背景色** | 白色 (#FFFFFF) | 确保内容清晰可读 |
| **布局模式** | 双模式切换 | 支持卡片式和列表式两种展示形式 |
| **字体** | 系统字体栈 | 优先使用 Inter，降级到系统默认字体 |

## 2. 设计规范

```

```
中性色系：
├── 深灰: #333333 (标题、正文)

├── 警告: #FF7D00 (预警、待处理)

## 6. 交互说明

### 6.1 卡片/列表切换交互

**切换流程**：
1. 用户点击右上角视图切换控件
2. 当前视图图标变灰，目标视图图标变君品绿
3. 内容区域平滑过渡到新布局（淡出淡入 300ms）
4. URL hash 更新（可选，用于刷新保持状态）

**交互状态**：
```
初始状态 (卡片视图):
  [列表]  [卡片]
   灰色    君品绿
          ────

点击切换到列表视图:
  [列表]  [卡片]
   君品绿   灰色
  ────

交互反馈:
- 点击时按钮有 0.95 缩放效果
- 内容区域 300ms 淡出淡入过渡
- 切换后保持当前筛选条件
```

### 6.2 君品绿交互反馈

| 交互类型 | 君品绿反馈 |
|---------|-----------|
| 按钮悬停 | 按钮背景变君品绿深色调 (#084C2C) |
| 按钮点击 | 0.95 缩放 + 君品绿波纹效果 |
| 链接悬停 | 文字颜色加深 + 下划线 |
| 输入框聚焦 | 边框变君品绿 |
| 行/卡片悬停 | 背景变君品绿浅灰 (#F0F7F4) |
