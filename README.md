# 项目一体化平台

## 项目概述

本项目是**贵州习酒销售公司信息部**的"项目一体化平台"前端原型，基于 PRD 和会议纪要的需求进行开发。

## 已完成功能 (Sprint 1)

根据需求评审会议纪要，第一阶段已实现以下核心功能：

### 1. 工作台 (Dashboard)
- 📊 统计卡片：进行中的项目、逾期任务、待处理预警、本月完成
- 📝 我的待办：按紧急程度排序，支持一键跳转
- ⚠️ 预警消息：风险预警展示
- 🏗️ 最近访问项目：快速进入最近查看的项目

### 2. 项目总览/台账 (ProjectList)
- 📋 项目卡片展示：项目编码、负责人、进度、预算
- 🔍 多维度筛选：项目类型、状态、部门
- 📊 卡片/列表双视图切换
- 🏷️ 快捷筛选标签

### 3. 立项管理 (Initiation)
- 📄 立项申请列表
- 🔍 申请详情查看
- ✅ 审批流程可视化
- ✍️ 审批操作（通过/驳回）

### 4. 数据中心 (Reports)
- 📈 核心指标统计
- 🏢 部门项目分布
- ⚠️ 风险预警汇总

### 5. 系统管理 (Settings)
- ⚙️ 组织架构、角色权限、用户管理入口
- 📊 系统信息展示
